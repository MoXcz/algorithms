El codigo recibe un valor n. Del intervalo [1,n] el codigo escribira fa si el numero es divisible entre 3, po si es divisible entre 5 y pofa si se cumplen ambas condiciones

    n=int(input("Ingresa un valor: "))
    for i in range (1, n+1):
        if i%3 == 0 and i%5 == 0:
            print("pofa")
        elif i%3 == 0:
            print("fa")
        elif i%5 == 0:
            print("po")
        else:
            print (i)
