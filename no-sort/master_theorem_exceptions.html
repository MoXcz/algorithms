Master Theorem Exceptions

  1. Recurrencia no recursiva: Si se tiene una función que no tiene una estructura recursiva, el método maestro no tiene sentido. Por ejemplo:
    2.             T(n) = 2n + 5
  2. Dependencia no matemática: Si la recurrencia involucra dependencias que no pueden expresarse en términos matemáticos simples, el método maestro no es aplicable. Por ejemplo:
    3.             T(n) = Tiempo actual + Tiempo de red
  3. Funciones no acotadas: Si la función no está acotada y no tiene un límite superior o no converge a un valor constante a medida que n crece, el método maestro no se puede aplicar. Por ejemplo:
    4.             T(n) = 2^n
